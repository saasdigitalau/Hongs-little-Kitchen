# 🍜 Hong's Little Kitchen Asian Cuisine

A complete, responsive restaurant website for **Hong's Little Kitchen Asian Cuisine** in Point Cook, Victoria.

## 📋 Overview

Single-page website built with plain HTML, CSS, and JavaScript — lightweight, fast, and production-ready.

## 🏗️ Structure

```
hongs-kitchen-website/
├── index.html         # Main single-page website
├── css/
│   └── style.css      # Complete responsive stylesheet
├── js/
│   └── main.js        # Interactivity, animations, form handling
├── chat-widget.js     # AI chatbot widget (by AI Agent Developer)
├── images/
│   ├── hero-food.jpg       # Hero background
│   ├── interior.jpg        # Restaurant interior
│   ├── dumplings.jpg       # Dumplings product shot
│   ├── prawn-toast.jpg     # Prawn toast product shot
│   ├── sweet-sour-pork.jpg # Sweet and sour pork product shot
│   └── logo.png            # Restaurant logo
└── README.md
```

## ✨ Features

- **Responsive design** — mobile, tablet, and desktop layouts
- **Warm Asian aesthetic** — rich reds, golds, dark accents
- **7 sections** — Hero, About, Menu, Reviews, Gallery, Location, Contact
- **Menu filtering** — category-based dish filtering
- **Interactive elements** — scroll animations, smooth navigation, mobile menu
- **AI Chat Widget** — embedded `<div id="ai-chat-widget">` for chatbot integration
- **Optimized images** — ~10MB → ~500KB (95% reduction), with SEO-friendly alt tags
- **Contact form** — functional form with subject selection
- **Google Maps embed** — interactive map for location finder
